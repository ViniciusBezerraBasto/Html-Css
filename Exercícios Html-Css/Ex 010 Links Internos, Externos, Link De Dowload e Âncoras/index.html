Links e Âncoras

Links Externos

Para você coloca um link externo no seu site você precisa de uma âncora ( local onde clica que abre o link ).
Para isso coloque o texto que você quer colocar como âncora para o link e envelope ele com
< a href="Seu link" target="_blank" rel="external" > Âncora para o sue link < /a >
Vejamos agora pronto
Âncora para o seu link

A parte do target="_blank" rel="external" serve para fazer com que o link se abra em uma guia externa do seu site

Links Internos

O Link Interno é um tipo de link que se usa para relacionar sua página com outras páginas suas. Uma forma de fazer isso é digitando a tag <a href="sua outra pagina"rel="next">voltar para a página anterior
Muito importante lembrar que a parte "rel="next" está falando para o google que o link que você irá clicar fará você avançar uma página
Já o "rel="prev" está falando para o google que você está voltando uma página Ex:

CLIQUE AQUI PARA AVANÇAR DE PÁGINA

Links Para Downloads

Você pode fazer também links de Downloads como por exemplo: Baixar pdfs, Baixar arquivos, Baixar apps e etc e para fazer isso faça da seguinte forma

  • Baixar Foco Milinário em PDF
  • Baixar Foco Milinário em ZIP

OBS: Quando o arquivo for em pdf acrescente a tag target="_blank" rel="external"
