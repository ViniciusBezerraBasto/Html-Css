Trabalhando com Listas

Listas Ordenadas

É o tipo de lista que organiza os passos de forma crescente do 1 ao infinito e você pode colocar qualquer passo entre os outros que ele organiza a numeração automaticamente basta usar o (ol) no começo e fim e o (li) em todos os passos. Do mesmo modo que exemplificarei abaixo 

            < ol >
                < li > Acordar 
                < li > Levantar da cama
                < li > Tomar café
                < li > Escovar os dentes
                < li > Limpar a pia
                < li > Tirar a carne do congelador
                < li > Fazer o almoço
                < li > Servir o almoço
                < li > Almoçar
                < li > Escovar os dentes
                < li > Usar o ccelular
                < li > Assitir live do Aruan
                < li > Jantar
                < li > Tomar um banho
                < li > Tomar cafézinho
                < li > Escovar os dentes
                < li > Abrir o curso de HTML5 do Gustavo Guanabara
                < li > Estudar
                < li > Guardar os materiais
                < li > Dormir
            </ ol >
Você pode também colocar tipos de enumeração dos passos, basta mudar a tag (ol)
< ol type="1" > < /ol > CASO QUEIRA EM ORDEM NUMERICA CRESCENTE
< ol type="A" > < /ol > CASO QUEIRA EM ORDEM ALLFABETICA MAIUSCULA
< ol type="a" > < /ol > CASO QUEIRA EM ORDEM ALFABETICA MINUSCULA
< ol type="i" > < /ol > CASO QUEIRA EM ORDEM DE NUMEROS ROMANOS MINUSCULOS
< ol type="I" > < /ol > CASO QUEIRA EM ORDEM DE NUMEROS ROMANOS MAIUSCULOS

    Lista não Organizada

    É um alista onde a ordem não importa ex: lista de compras
    Para realizar essa lista usamos as seguintes tags:< ul > e <li>. Lembrando que a tag < ul > tem suas variações como por exemplo:
    < ul type="circle"> Que coloca os passos da lista após um circulo vazio como no exemplo abaixo

    • Comprar feijão
    • Comprar Arroz
    • Comprar Farinha
    • Comprar suco

    Lembrando que o < ul type="circle"> já vem insirido padrão na tag < ul > como no exemplo abaixo
    • Comprar feijão
    • Comprar Arroz
    • Comprar Farinha
    • Comprar suco

    < ul type="square"> Que coloca os passos da lista após um quadrado como no exemplo abaixo
    • Comprar feijão
    • Comprar Arroz
    • Comprar Farinha
    • Comprar suco

    < ul type="disc"> Que coloca os passos da lista após um circulo preenchido como no exemplo abaixo
    • Comprar feijão
    • Comprar Arroz
    • Comprar Farinha
    • Comprar suco

    Juntando Listas

    Lista A + Lista A

    Para misturar listas do mesmo tipo você pode fazer da meneira abaixo ↓
    Mas para fazer isso precisa deixar desse modo 

                Idiomas
            <ol type="I">
                 <li>Europeus 
                 <ol type="a">
                    <li>Alemão
                    <li>Francês
                    <li>Italiano 
                </ol>
                <li>Americanos
                <ol type="a" start="4">
                        <li>Inglês
                        <li>Espanhol
                </ol>
                <li>Asiáticos
                <ol type="a" start="6"> 
                        <li>Japones
                </ol>
            </ol>

    Obs: Os (starts) são usados para dar continuidade na ordem por exemplo se o ultimo (li) foi o C para começar do D no próximo tem colocar start= "4". 4 porque é o D é o quarto elemento do alfabeto
    Agora veremos de forma pronta abaixo
    Ex :
    Idiomas
    1. Europeus
      1. Alemão
      2. Francês
      3. Italiano
    2. Americanos
      1. Inglês
      2. Espanhol
    3. Asiáticos
      1. Japones

    Listas A + Listas B

    Para misturar as listas diferentes não tem segredo é somento colocar (ul) aonde quiser de forma livre os passos Ex: 

                Idiomas
            <ol type="I">
                 <li>Europeus 
                 <ul type="circle"> 
                    <li>Alemão
                    <li>Francês
                    <li>Italiano 
                </ul>
                <li>Americanos
                <ul type="circle"> 
                        <li>Inglês
                        <li>Espanhol
                </ul>
                <li>Asiáticos
                <ul type="circle"> 
                        <li>Japones
                </ul>
            </ol>

    Agora veremos de forma pronta abaixo Ex:
    Idiomas
    1. Europeus
      • Alemão
      • Francês
      • Italiano
    2. Americanos
      • Inglês
      • Espanhol
    3. Asiáticos
      • Japones
