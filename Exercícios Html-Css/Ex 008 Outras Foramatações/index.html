Código Fonte

O comando <code> é usado muitas vezes em codigos (por isso o nome) e ele deixa o tamanho do espaçamento de cada letra de forma igualada
EX (sem <code>) : document.getElementById('teste')
Perceba que as letras "t","i" e "l" estão sendo quase que eesmagadas pelas letras de sua tras e sua frente. Isso é resolvido com a tag <code>
EX (com <code>) : document.getElementById('teste')
Perceba que dessa vez as letras "i","t"e"l" tiveram o mesmo espaçamento que as outras possuiam

Pré Formatação

A Pré Formatação está presente no html através da tag < pre >ela é usada para deixar os finais e começos de linhas do mesmo modo que você escreveu no VS CODE
Ela quase sempre é utilizada juntamente com a tag < code >,como por exemplo abaixo 

    
    < pre >
        < code >
Agora para você entender como funciona a tag < pre > veja os exemplos abaixo
EX:(Sem o < pre >) def contador (i , f): c = i While c = f: c + = 1 print(c , end='') contador (1 ,10)

Perceba que as linhas se juntaram sem respeitar o começo e fim que eu coloquei no VS CODDE. Vamos ao exemplo que utilizamos a tag < pre >
EX: 

        
            def contador (i , f):
                c = i
                While c = f:
                    c + = 1
                    print(c , end='')
            contador (1 ,10)

Para você tirar o espaço que fica no começo da página até o primeiro codigo selecione todos os codigos, aperte SHIFT + TAB até encostar na borda da página.
Para fazer o inverso (se distanciar da borda da página) selecione todos os codigos e aperte TAB até a posição desejada. Veja os Exemplos abaixo
EX (tirando o espaço até a borda): 

            
def contador (i , f):
    c = i
    While c = f:
        c + = 1
        print(c , end='')
contador (1 ,10)

Perceba que o começo da linha do primeiro e ultimo codigo encostou na borda da página
Veja agora o exemplo utilizando somente o TAB para distanciar mais da borda da página
EX(acrescentando espaço até a borda): 

        
                                def contador (i , f):
                                    c = i
                                    While c = f:
                                        c + = 1
                                        print(c , end='')
                                contador (1 ,10)

Citações Simples

Para fazer citações de formas simple use a tag < q >(ela deixará o texto selecionado entre aspas)
EX: Como dizia meu amigo Nunca pule etapas.
Quando for fazer uma citação utilize a tag < q > e não simplesmente colocar entre aspas, pois caso alguem quando ver seus codigos vai entender de cara que as aspas significam citação

Citações Completas

Para realizar citações completas utiliza-se a tag < blockquote >
EX:Como dizia Stuart Firestein no livro:Ignorãncia: Como ela impulsiona a ciência

O método científico é observação, hipótese, manipulação, mais observação e novas hipóteses, realizando num loop infindável de descobertas

Dá também para você colocar de onde você tirou essa citação escrevendo < blockquote cite="link de onde você tirou a citação"> A citação </blockquote >
Isso não irá aparecer no site, mas serve para o google saber de onde você tirou essa informação

Abreviação

A formatação da Abreviação serve para que quando o usuário passa o ponteiro do mouse em cima de determinada vairação apareça o significado dela.
Para fazer isso se utiliza a tag < abbr title = "significado da abreviação" > A abreviação </ abbr >
EX: A quant de carne está pouca para a festa no nosso ap.

Texto Invertido

Para fazer um texto invetido utilize a tag < bdo dir="rtl">O texto </ bdo >
EX: Nosso Mundo
