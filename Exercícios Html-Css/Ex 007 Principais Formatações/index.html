Principais Formatações

Negrito ou Destaque

Você pode colocar determinado trecho do texto em negrito usando a tag < b >(que é uma tag de forma / estilo) EX: O Pedro pulou da cama

Você também pode usar < strong > que é uma tag de semantica
Use mais a Tag < strong >

Itálico ou Ênfase

Você pode colocar determinado trecho do testo em itálico usando a tag < i >(que é uma tag de forma/estilo) EX: O Pedro pulou da cama

Você também pode usar a tag < em > que é uma Tag de semântica
Use mais a tag < em >

Texto Marcado

Para marcar um texto você tem que utilizar a tag Mark
Porém esta tag só funciona se você apertar: ALT + W selecionar o trecho e digitar MARK e der ENTER EX: O pedro pulou da cama e caiu de cabeça

O MARK você pode mudar a cor dele de dois modos um é você mudando abaixo da tag <Title> colocando <style> embaixo dessa linha coloque mark{ e na outra linha coloque background-color:(Aqui escolha a sua cor para todos os MARK que tiverem) e na linha de baixo coloque o } (LEMBRANDO ISSO JÁ FAZ PARTE DE CSS E NÃO DE HTML)

Você tem a opção de mudar somente um MARK em especifico como por exemplo o MARK abaixo
EX: O Pedro pulou da cama e caiu de cabeça

E para eu conseguir fazer isso tive que usar a tag <mark style="background-color: A Cor que quiser colocar;">O Texto que quiser marcar</mark>

Textos Grandes e Textos Pequenos

Para colocar um texto na forma grande antes era utilizado a tag <big> MAS essa tag até que funciona MAS não é bom voê usa-la pois o html pode tirar ela do ar qualquer momento e também se você usa-la o google ele não vai recomendar da maneira certa seu site pois está usando tags ultrapassadas
Mas você pode colocar o texto de forma maior utilizando o CSS
EX: O Pedro pulou da cama

É possivel você colocar um texto de forma pequena utilizando o <small> (Essa tag é uma tag que o html aceita sem você precisar ter medo de ele remover qualquer hora)
Ex: O Pedro pulou da cama

Texto Deletado e Texto Inserido

Para colocar um texto no modo deletado (também chamado de forma equivocada de "texto riscado") você precisa colocar o texto entre a tag <del>
EX: O Pedro pulou da cama

Já para colocar o texto em modo inserido (também chamado de forma equivocada de "texto sublinhando") você precisa botar o texto entre a tag <ins>
EX: O Pedro pulou da cama

Texto Sobrescrito e Subscrito

Você para colocar um texto de forma sobrescrita (em cima) você tem que utilizar a tag <sup>
EX: X27 + 16

E para inserir um texto de forma subscrita (em baixo) se usa a tag <sub>
EX: H2O
